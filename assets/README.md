# 🚀 Impulsa Pro | Landing Page Profesional

Landing page moderna y responsiva desarrollada con HTML, CSS y JavaScript puro, enfocada en ofrecer soluciones digitales para negocios.

## 🌐 Descripción

Impulsa Pro es una página web tipo landing diseñada para presentar servicios digitales de manera clara, moderna y atractiva.  
Incluye animaciones suaves, carrusel dinámico y diseño adaptado a dispositivos móviles.

El objetivo del proyecto es demostrar habilidades en:

- Maquetación profesional
- Organización estructural de archivos
- Animaciones CSS
- Manipulación del DOM con JavaScript
- Buenas prácticas de desarrollo frontend

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3 (Flexbox + Grid + Transiciones)
- JavaScript Vanilla
- Google Fonts (Poppins)
- Font Awesome

---

## 🎯 Funcionalidades

✔ Navbar fija con efecto glass  
✔ Hero con imagen de fondo y overlay profesional  
✔ Carrusel automático con controles manuales  
✔ Animaciones al hacer scroll  
✔ Secciones organizadas por bloques  
✔ Botón flotante de WhatsApp  
✔ Diseño responsive
